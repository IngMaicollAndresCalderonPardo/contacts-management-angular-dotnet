Bienvenido a Contact-App

Esta aplicación es un desafío técnico que implementa un backend en .NET Core y un frontend en Angular. Permite gestionar contactos: crear nuevos, listar, buscar y eliminar.

Pestañas disponibles:

  • Register: Formulario para agregar nuevos contactos con validaciones.
  • Contacts: Lista de todos los contactos con opción para eliminar o editar.
